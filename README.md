# Examples of mocking Context Managers with pytest

# Geeting the code 

```shell
git clone https://github.com/rlaboiss/pytest-mock.git
cd pytest-mock
```

# Running the tests

Just do:

```shell
make
```

or run the individual tests manually:

```shell
pytest-3 test_run_command.py
pytest-3 test_write_file.py
```
